# 🎬 Netflix Content Analytics Dashboard

## 📌 Project Overview
This project analyzes Netflix content using Power BI.

## 📊 Features
- Content distribution by type (Movies vs TV Shows)
- Ratings analysis
- Release year trends
- Drill-through single title view
- Dynamic narrative insights

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- GitHub

## 📂 Dataset
Netflix Movies and TV Shows Dataset

## 🚀 Key Insights
- Movies dominate Netflix catalog.
- Most content added after 2015.
- TV Shows have higher average duration consistency.
